# Security Policy

## Scope

This repository manages AWS Organization governance controls including Service Control
Policies (SCPs). Changes to this code directly affect security boundaries across
AWS accounts.

## Reporting a Vulnerability

If you find a security vulnerability in any SCP policy, Terraform configuration,
or CI/CD workflow, please **do not open a public GitHub Issue**.

Instead, report it privately via email: <alejandrogarcia@iteso.mx>

Include:

- The affected file or resource
- A description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact on AWS accounts

## Response

Reports will be reviewed and addressed promptly. Security issues in governance
controls are treated as high priority.
